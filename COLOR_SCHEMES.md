# Варианты цветовых схем

## ✅ Вариант 1: Энергия и рост (ПРИМЕНЁН)

**Характеристики:**
- Яркий зелёный (#10B981) как основной цвет
- Янтарный (#F59E0B) для акцентов
- Светло-зелёный фон (#F0FDF4)
- Вызывает ощущение роста, развития, энергии

**Цвета:**
- Primary: #10B981 (яркий зелёный)
- Accent: #F59E0B (янтарный)
- Base: #F0FDF4 (светло-зелёный)
- Heading: #065F46 (тёмно-зелёный)

**Эмоции:** Энергия, рост, развитие, оптимизм

## Вариант 2: Доверие и профессионализм

**Характеристики:**
- Синий (#3B82F6) как основной цвет
- Оранжевый (#F97316) для акцентов
- Светло-голубой фон (#EFF6FF)
- Вызывает доверие, профессионализм

**Цвета:**
- Primary: #3B82F6 (синий)
- Accent: #F97316 (оранжевый)
- Base: #EFF6FF (светло-голубой)
- Heading: #1E40AF (тёмно-синий)

**Эмоции:** Доверие, профессионализм, надёжность

## Как применить другой вариант

Откройте `tailwind.config.ts` и замените значения в секции `colors`:

```typescript
colors: {
  base: '#EFF6FF', // для варианта 2
  primary: {
    DEFAULT: '#3B82F6', // для варианта 2
    // ... остальные оттенки
  },
  accent: '#F97316', // для варианта 2
  // ...
}
```

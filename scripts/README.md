# Скрипты конвертации

## Конвертация Excel в JSON

Для конвертации Excel файлов с тестами в JSON формат используйте скрипт `convert_excel_to_json.py`.

### Требования

```bash
pip install openpyxl
```

### Использование

```bash
python scripts/convert_excel_to_json.py
```

Скрипт автоматически:
1. Читает Excel файлы из папки `tests/FREE/`
2. Конвертирует их в JSON формат
3. Сохраняет в `src/tests/FREE/`

### Структура Excel файла

Excel файл должен содержать следующие колонки:
- **Колонка A**: Номер вопроса (1-5)
- **Колонка B**: Текст вопроса
- **Колонка C**: Значение варианта 1 (P, J, E, I, T, F, N, S)
- **Колонка D**: Текст варианта 1
- **Колонка E**: Значение варианта 2
- **Колонка F**: Текст варианта 2

### Пример структуры

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| 1 | Текст вопроса 1 | P | Описание P | J | Описание J |
| 2 | Текст вопроса 2 | E | Описание E | I | Описание I |
| ... | ... | ... | ... | ... | ... |

### Результат

После выполнения скрипта будут созданы файлы:
- `src/tests/FREE/12-17.json`
- `src/tests/FREE/18-20.json`
- `src/tests/FREE/21plus.json`

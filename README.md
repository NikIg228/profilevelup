# ProfiLevelUp — Платформа профориентационного тестирования

Комплексное решение для психологического тестирования и профориентации, включающее веб-интерфейс, REST API, систему генерации PDF-отчетов и автоматизированные воркеры.

## 🏗 Архитектура системы

Проект реализован на базе современной микросервисной архитектуры с разделением ответственности:

1.  **Frontend (prof_oprosnik_v2)**: SPA на React + Vite, обеспечивающее интерактивное прохождение тестов и личный кабинет.
2.  **Backend API (pb_backend)**: FastAPI сервис для управления данными, валидации ответов и интеграции с платежами.
3.  **Workers Pool**: Набор асинхронных воркеров на Python для тяжелых задач (генерация PDF, рассылка почты).
4.  **Infrastrucure**:
    *   **Supabase**: База данных (PostgreSQL), Хранилище (Storage) и Авторизация (Auth).
    *   **Upstash (Redis)**: Очередь задач и кэширование.
    *   **Resend (SMTP)**: Сервис отправки электронных писем.

## 🚀 Технический стек

### Frontend
- **Ядро**: React 18, TypeScript, Vite.
- **Стилизация**: Tailwind CSS.
- **Анимации**: Framer Motion, GSAP, Swiper.
- **Состояние**: Zustand, React Router DOM v6.
- **API**: Supabase JS SDK, Zod (валидация).

### Backend & Infrastructure
- **Ядро**: Python 3.10+, FastAPI, Uvicorn.
- **База данных**: PostgreSQL (через Supabase SDK).
- **Очередь задач**: Redis (Upstash) + кастомная реализация воркеров.
- **Генерация документов**: python-docx + LibreOffice (DOCX -> PDF).
- **Валидация**: Pydantic v2.
- **Логирование**: Structlog.

## 🌟 Основные возможности

- **3 типа тестов**: Free (5 вопросов), Extended (28 вопросов), Premium/VIP (расширенный анализ).
- **Автоматическая генерация PDF**: Создание детализированных отчетов на основе психологического профиля.
- **Платежная система**: Интеграция с Robokassa.
- **Личный кабинет**: История тестов, управление профилем, доступ к купленным отчетам.
- **Мониторинг**: Эндпоинты для проверки состояния системы (CPU, RAM, доступность БД/Redis).
